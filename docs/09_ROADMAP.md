# Balatro Blast — Roadmap

## Current Version: 1.0.0 (MVP)

### MVP Features ✅
- [x] Core poker hand evaluation (all 10 hand types)
- [x] Scoring engine with chip × mult formula
- [x] 20 Joker cards with distinct effects
- [x] 8 antes with escalating blind targets
- [x] Shop with Jokers, Tarot, Planet cards
- [x] Canvas-rendered playing cards (no assets)
- [x] Dark neon visual theme
- [x] Save/load via Hive
- [x] Statistics tracking

## Version 1.1 — Polish & Content

- [ ] **Card Animations**: Smooth deal, play, and discard animations using Flame effects
- [ ] **Score Pop**: Animated chip/mult counters that fly up on scoring
- [ ] **Sound Effects**: Card flips, scoring jingles, shop music
- [ ] **More Jokers**: Expand to 40+ jokers (matching Balatro's roster)
- [ ] **Card Enhancements**: Foil, Holographic, Polychrome editions
- [ ] **Boss Blind Effects**: Implement all boss blind restrictions
- [ ] **Blind Skip**: Allow skipping blinds for money bonus

## Version 1.2 — Deck Building

- [ ] **Card Editions**: Gold, Steel, Glass, Lucky card types from Tarot
- [ ] **Spectral Cards**: Rare consumables with powerful deck-altering effects
- [ ] **Vouchers**: Permanent upgrades purchasable in shop
- [ ] **Deck Viewer**: Browse all cards in current deck
- [ ] **Seed Runs**: Enter a seed for reproducible runs

## Version 1.3 — Meta Progression

- [ ] **Stakes System**: Difficulty levels (White through Gold stake)
- [ ] **Collection**: Track discovered jokers, cards, tarots
- [ ] **Achievements**: Unlock achievements for specific accomplishments
- [ ] **Run History**: Review past runs with stats
- [ ] **Daily Challenge**: Same seed for all players each day

## Version 2.0 — Multiplayer & Online

- [ ] **Leaderboards**: Submit scores to global leaderboard
- [ ] **Challenge Runs**: Share custom challenge configurations
- [ ] **Spectator Mode**: Watch other players' runs
- [ ] **Tournament Mode**: Bracket-style competition

## Version 2.1 — Platform Expansion

- [ ] **Web Build**: Deploy to web via Flutter Web
- [ ] **Desktop Support**: Windows/macOS/Linux builds
- [ ] **Tablet Optimization**: Larger layout for iPad/Android tablet
- [ ] **Controller Support**: Gamepad input support

## Technical Debt & Improvements

- [ ] **Comprehensive UI Tests**: Widget tests for all screens
- [ ] **Integration Tests**: Full game loop integration tests
- [ ] **Performance Profiling**: Optimize rendering for 60fps on low-end devices
- [ ] **Accessibility**: Screen reader support, colorblind modes
- [ ] **Localization**: i18n support for multiple languages
- [ ] **CI/CD Pipeline**: Automated builds and test runs on GitHub Actions
